# miChecker research source pack

This directory preserves the official April 2024 FAQ PDF and a machine-extracted Markdown copy for provenance and review.

- Official landing page: https://www.soumu.go.jp/info-accessibility-portal/webaccessibility/michecker/
- Official PDF: https://www.soumu.go.jp/info-accessibility-portal/assets/documents/webaccessibility/michecker/faq.pdf
- Local original: `faq-2024-04.pdf`
- Local extraction: `faq-2024-04.extracted.md`
- Integrity and related public manuals: `source-record.json`

The reusable skill does not depend on miChecker, parse its output, or treat its messages as WCAG or JIS outcomes. The source material is used to derive tool-independent screening patterns and evidence-handling practices. Every candidate mapping must still be checked against the current W3C and WAIC sources and confirmed by a person for the scoped target.
